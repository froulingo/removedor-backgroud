# removedor-de-fondos-img


## ⚙️ Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/froulingo/removedor-backgroud.git
```

2. Crea el entorno virtual
```bash 
python -m venv env
```

3. activa el entorno virtual (windows)
```bash
./env/Scripts/activate
```

4. instalar las dependencias
```bash
pip install -r requirements.txt
```

5. ejecutar el main.py de esta manera:
```bash
uvicorn main:app --reload
```

6. Abre tu navegador en la url que te dió tu terminal
